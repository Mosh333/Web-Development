Part 1 - BookIt - The Online Bookmarker

  Part 1 asks for an implementation of an online bookmarking service, which allows user to save their favourite links online. Before log in, the app should show up to top 10 most popular links based on the database.

This app should allow user to register, login, and logout. While logged in, the system should allow user to add, edit, and delete any website links. Through Javascript, the URL link input by user must be validated.

There are two MySQL tables used to implement this app:

  • UserList: username (Primary Key), passwd
  • Bookmark: LinkID (Primary Key), urlLink, user (foreign key to username in UserList)

Technologies used are HTML, CSS, Javascript, and PHP.

Part 2 - Educord - The Online Learning Management System

  Part 2 asks for an implementation of an online learning system, which allows instructors to deliver their courses online to students. Before log in, the app should show all educord online courses uploaded into the database.

This app should allow user to register, login, and logout. While logged in, the system should allow instructors to add, edit, and delete any course materials. Students can view the course, do quizzes, upload assignments, and look at references, as well as enroll or withdraw from courses.

There are several MySQL tables used to implement this app:

  • educordUserList
  • educordAnnouncements
  • educordModules
  • educordReferences
  • educordAssignments
  • educordQuizQuestions
  • educordCourseContent
  • educordEnrolledStudents
  • educordAssignmentSubmissions
  • educordQuizGrades

Technologies used are HTML, CSS, Javascript, and PHP.
